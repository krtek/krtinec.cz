## Freelo

Z českých luhů a hájů pochází [Freelo](https://www.freelo.cz/) – konkurence populárnímu Basecampu.

Freelo se používá podobně jako Trello, ale
navíc nabízí funkce sledování času a stažení výkazů.

Pro malé projekty zadarmo, pro větší za 400 měsíčně.

## Document CSS

[Document CSS](http://documentcss.com/index.html) slouží k dokumentaci
CSS souborů přímo v kódu:

```css
/**
    * @stylesheet buttons.less Buttons
    * @parent Styles.baseline-elements
    *
    * @description
    * All defined button styles and states belong here.
    **/
button,
.button {
	background-color: @colorLinks;
	border: 1px solid darken(@colorLinks, 10%);
	color: @clear;
}
```

## Snyk

Používáte NodeJS? Pokud ano, tak vaše aplikace obsahuje dost pravděpodobně bezpečnostní problém.
Proč? Protože 14% npm balíčků obsahuje známou zranitelnost.

[Snyk](https://snyk.io/) je nástroj, který zkontroluje vaší aplikaci,
najde problémové balíčky a doporučí řešení.
